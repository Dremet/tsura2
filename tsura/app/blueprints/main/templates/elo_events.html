What is Elo?  Originally invented for chess by Arpad Elo, this rating system translates game outcomes into changes in a single number that represents skill.

How does it work for multi‑player races?  We adapt the chess formula: each driver’s current rating yields an expected finishing position. The bigger the gap between expectation and reality, the larger the Elo gain or loss.

Update cycle.  Ratings update automatically a few minutes after every Event‑server race.  If you skip a race, your rating stays unchanged.

Limitations.  Elo needs several results to stabilise, so this table lists only drivers with at least three participations. Car‑specific skill and other nuances are not yet separated – it’s an approximate but useful indicator.
